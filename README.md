# misc-public

Public projects and demos - deployed via GitHub Pages

## Projects

### [Pocket Heist](pocket-heist/)

Ein asymmetrisches 2-Spieler Stealth-Strategiespiel. Spieler 1 baut als Architekt ein Level mit Wachen, Kameras und Fallen. Spieler 2 versucht als Infiltrator, den Tresor zu erreichen.

**[Play](https://sxxs.github.io/misc-public/pocket-heist/)** | v2.4.2

- Mobile-first PWA (installierbar)
- Auto-Skalierung auf alle Bildschirmgrößen
- Level-Codes zum Teilen
- Replay-System
- Generative Musik mit Tone.js

### [Multiplication Troll Game](multiplication-troll-game/)

Retro-Style Multiplizierungsspiel zum Üben des kleinen 1x1 mit fiesen Troll-Mechaniken.

**[Play](https://sxxs.github.io/misc-public/multiplication-troll-game/)** | v1.2.0

- Aufgaben ändern sich spontan beim Tippen
- Retro-Pixel-Grafik mit Strichmännchen
- Mobile-optimiert mit On-Screen Keyboard
- PWA (installierbar)

### [WIAI25 Enhance](wiai25-enhance/)

Static demo for WIAI 25 Collective Enhance project.

**[View](https://sxxs.github.io/misc-public/wiai25-enhance/)**
