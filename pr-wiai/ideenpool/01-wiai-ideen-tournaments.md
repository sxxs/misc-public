# WIAI Social Content Backlog — Ideen-Tournaments

Für jeden Post-Anlass: 8 Varianten im Tournament. Jede Variante ist ein vollständiger Post (2–4 Slides). Stimme: süffisant, lakonisch, Mic Drop am Ende. WIAI-Nennung organisch eingebaut. Keine CTA-Slides.

Bewertungskriterien:
- **Teilbar?** Würde jemand das an Freunde schicken?
- **Insight?** Nimmt der Viewer etwas mit?
- **Mic Drop?** Sitzt der letzte Satz?
- **Organisch?** Wirkt die WIAI-Nennung natürlich?
