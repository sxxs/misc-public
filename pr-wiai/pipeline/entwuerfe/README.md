## Tag-Legende

- **#ok** — Freigegeben, organischer Post
- **#ad** — Fuer bezahlte Kampagnen, direkter Ton
- **#rewrite** — Umgeschriebene Version (Text aus Rewrite-Dateien 11-14)
- **#ja** — Besonders gut (behandelt wie #ok)
- **#ton-check** — Tonfall-Problem identifiziert, Rewrite darunter

## Post-Elemente

- **S1, S2, S3** — Slides 1-3 (Carousel/Video)
- **Button** — Optionaler vierter Beat, klein/gedimmt, selbstironisch
- **Uebrigens** — Beilaeufige Factoid, Monospace, am unteren Rand
