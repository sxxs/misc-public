101 · Plotter-Komponist

Komponiere eine Tonfolge — Klick schaltet jede Stufe weiter:
▬ malen = tiefer Ton (Stift zeichnet, langsam) · leer = hoher Ton (Leerfahrt, schnell) · ● Klack = Stift tippt (Punkt + Klick).

Aus deiner Komposition entsteht ein bildfüllendes Muster: Die Tonfolge wird auf einer Schlangenlinie (oder Spirale) ausgelegt. Punkte am Zeilenanfang/-ende „verankern" den Pfad, damit der Plotter wirklich die komponierte Strecke fährt (sonst nimmt er die Abkürzung) — und so genau deine Musik erzeugt. Der rote Stift fährt beim Anhören live mit.
