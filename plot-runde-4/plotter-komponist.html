101 · Plotter-Komponist

Komponiere eine Tonfolge — Klick oder Tastatur:
▬ malen = Stift zeichnet (tiefer Ton) · leer = Leerfahrt (hoher Ton) · ● Klack = Stift tippt

D Malen   Space/T Leer   K/. Klack   ←→ Cursor   ⌫ Zurück+Leer

Aus deiner Komposition entsteht ein bildfüllendes Muster: Das Motiv wird auf einer Hilbert-Kurve ausgelegt — alle Abschnitte gleich lang, alle achsen-parallel (kein Singen). Jedes Segment ist rein horizontal (X-Motor, links) oder rein vertikal (Y-Motor, rechts). Punkte verankern den Pfad an Motivgrenzen.
